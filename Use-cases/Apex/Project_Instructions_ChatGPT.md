# 📘 Project: Apex — Wilderness Marketing Intelligence System

## Purpose
Guide is a Wilderness-first marketing intelligence engine that automatically diagnoses paid media performance, reveals hidden growth opportunities, and creates leverage across PPC operations — starting with Botswana.

It is the first prototype of a system that can later scale across Jacada, Yellow Zebra, and eventually the entire B2C portfolio.

## Strategic Intent
- Replace static dashboards with dynamic diagnostics
- Codify Gareth's operating intelligence into reusable systems
- Enable performance clarity without requiring Gareth to run every analysis
- Impress internal stakeholders (Keith, Hadley) with self-evident utility and IP potential
- Set the foundation for a potential SaaS or licensing model post-transformation

## Phase 0 MVP: “Botswana Opportunity Radar”
- Run 12 automated PPC diagnostics against Botswana campaigns
- Output actionable recommendations overnight
- Summarize findings in plain English + data tables
- Prove system value without Gareth needing to lift a finger

## Key Questions This Project Must Answer
- Where are we missing ROAS in Botswana?
- Which clusters are underexploited?
- Where are we bleeding in the funnel?
- What would Gareth do — if he had 10x more time?
- What happens when this runs across all regions and brands?

## Deliverables
- Daily auto-run diagnostics across PPC data
- Output: diagnostics_summary.md + opportunity_matrix.csv
- Interactive demo (Streamlit or notebook)
- Modular Python scripts per diagnostic
- Playbook for expansion across markets

## Constraints
- Local-only or secure VM use preferred
- Avoid cloud lock-in unless necessary
- Must integrate easily with GA4, Google Ads exports, CRM exports
- Deliver high trust / high clarity output

## Owner
Gareth (Digital Director, Wilderness)

## Intended Audience
- Internal: Hadley (CCO), Digital Team, Keith (CEO)
- Future: Jacada, Yellow Zebra, other execs
