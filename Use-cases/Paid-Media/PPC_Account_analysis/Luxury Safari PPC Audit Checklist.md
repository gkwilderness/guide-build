# Luxury Safari PPC Audit Checklist

## I. Account Structure

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Segmentation by intent | Separate high-intent (e.g. “luxury Botswana safari”) from TOF |  |
| Brand vs Non-Brand separation | Clearly separated, with different budgets & KPIs |  |
| Geo-targeting refined | Top-tier countries only, excluding unqualified regions |  |
| Device targeting | Optimized for HNW browsing behavior (tablet/desktop bias) |  |
| Campaign naming clarity | Clean, strategic, and reflects funnel stage |  |

## II. Keyword Strategy

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Intent hierarchy used | "Luxury safari Botswana" > "Botswana safari" > "African safari" |  |
| Negative keyword lists applied | To block budget waste (e.g. “cheap,” “volunteer,” “Kruger day trip”) |  |
| Exact/Phrase match mix | Match types used strategically — not all broad |  |
| Competitor conquesting assessed | Buying against other luxury operators? |  |

## III. Ad Copy & Creative

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| HNW language & tone | No “best deal” or “budget-friendly” messaging |  |
| USPs clearly stated | Eco impact, exclusivity, private guides, conservation impact |  |
| Emotion-driven messaging | Speaks to identity, legacy, restoration |  |
| Use of rich ad extensions | Sitelinks, structured snippets, callouts |  |
| Performance creative tests running | Multiple headlines + descriptions actively tested |  |

## IV. Landing Pages

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Destination-specific LPs | Each major region or camp has its own high-converting page |  |
| Ultra-luxury visual cues | Full-screen visuals, aspirational tone, high production |  |
| Minimal form friction | No multi-step forms unless value-justified |  |
| Trust indicators | Awards, expert reviews, security badges, conservation credentials |  |
| Speed optimized | Loads in <3s on mobile and desktop |  |

## V. Budget & Bidding

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Budget aligned with intent value | High-intent terms get priority budget |  |
| Bidding strategy is informed | Blend of Smart Bidding + manual tuning where needed |  |
| Retargeting audience built | Warm traffic (30–90 days) is captured and retargeted |  |
| Time-of-day bid modifiers | Evening/weekend browsing patterns optimized |  |

## VI. Attribution & Measurement

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Lead quality tracked (not just volume) | Offline conversion uploads to link lead quality to spend |  |
| UTM tagging consistent | UTMs structured and correctly deployed across all campaigns |  |
| Channel-assisted conversions reviewed | PPC evaluated beyond last-click model |  |
| Post-lead funnel visible | Tracking leads through to booking outcome clearly |  |

## VII. Strategic Fit

| Checkpoint | Criteria | Pass/Fail |
|------------|----------|-----------|
| Aligned with commercial priorities | Focused on filling priority camps, routes, dates |  |
| Scaling roadmap exists | Clear view of budget elasticity and scaling points |  |
| Test plan defined | 3–5 priority tests per quarter with owners and deadlines |  |
| Redundancies removed | No zombie campaigns, minimal broad-match waste |  |

---

# Scoring Guide

- **<70% Pass:** High risk. PPC is leaking money and missing opportunity.
- **70–85% Pass:** Functioning, but not optimized for luxury growth.
- **85–100% Pass:** Elite structure. Ready to scale confidently.

---

# Operating Principles

- Focus on **impact per click**, not volume.
- Prioritize **intent**, not just traffic.
- Build for **quality leads and bookings**, not CTR vanity.
- Align **strategy with the luxury buyer journey** — emotional first, rational second.
