## Growth Roadmap — Luxury Safari PPC Scale Plan

### Week 1: Conversion Signal Audit + Scoring Design
- Map CRM lead outcomes to conversion quality tiers
- Design lead score thresholds for import
- Create test imports manually if needed

### Week 2: Offline Conversion Setup & Campaign Restructure
- Setup Offline Conversion Import (OCI) or CRM-based proxy labels
- Restructure campaigns by funnel stage and match type
- Allocate budgets aligned with funnel intent

### Week 3: Launch BOFU/MOFU Split Campaigns
- Launch Exact and Phrase Match campaigns by geo or camp
- Apply audience lists (GA4, in-market, remarketing)
- Build shared negative keyword lists to prevent cannibalisation

### Week 4: RSA + Landing Page Variant Build & Launch
- Write 5–10 Responsive Search Ad (RSA) variations by funnel
- Launch 2–3 LP variants per funnel tier (BOFU = Booking, MOFU = USP, TOFU = Content)
- Route traffic based on funnel intent

### Week 5: Broad Match TOFU Test with Audience Layering
- Launch tightly controlled Broad match campaign
- Limit with in-market and GA4 audiences
- Set CPC cap; monitor search term reports closely

### Week 6: Performance Review + Budget Scaling
- Review CPL/CVR/ROAS by funnel stage and match type
- Scale budget into high-performing segments
- Pause/redirect underperforming traffic
