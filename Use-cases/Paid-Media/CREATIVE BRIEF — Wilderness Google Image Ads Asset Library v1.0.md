### 📝 Project Overview  
This project will produce a centralised image asset library optimised for all major Google Ads visual placements.  
Assets must reflect Wilderness' luxury positioning, conservation focus, and distinct persona journeys — while aligning to Google’s format specifications.

---

### 👥 Audience  
- High-net-worth travellers  
- Luxury adventure seekers  
- Safari-first timers & legacy returners  
- US, UK, DE priority markets

---

### 🎯 Key Objectives  
- Deliver a fully tagged, usage-ready image asset library  
- Support all ad formats used in Search, Performance Max, Discovery, and Display  
- Enable creative rotation and automated ad personalisation  
- Reflect brand tone: luxury, wilderness, privacy, awe

---

### 📡 Channels  
- Google Search (Image Extensions)  
- Performance Max  
- Discovery Campaigns  
- Responsive Display Ads (RDA)  
- YouTube Home Feed (via PMax/Discovery)

---

### 🧠 Key Considerations  
- No text overlays, filters, or visual clutter  
- Emotion-forward, high-impact imagery  
- Must pass Google ad policy (no excessive graphic scenes or alcohol)  
- Use real Wilderness experiences and locations only  
- All images must be rights-cleared for paid usage  

---

### 📌 Mandatories  
- Deliver all sizes & ratios listed  
- Minimum 3 variations per combination  
- Final export: Web-ready (JPG/PNG) + Source files (PSD or RAW)  
- Folder naming structure must match the tagging framework  
- Include production tracker with metadata per image  

---

### 📦 List of Deliverables by Format

#### 🟩 Search Ads (Image Extensions)
- 5x Square (1:1) images  
- 5x Landscape (1.91:1) images  

#### 🟦 Performance Max

- 5x Square (1200x1200)

- 5x 1200x628 thumbnails  
- 3x Lifestyle hero stills for homepage feed presence
